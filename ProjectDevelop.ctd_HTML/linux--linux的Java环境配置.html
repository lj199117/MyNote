linux的Java环境配置

1. 安装jdk
• 解压
tar -zxvf jdk1.7.0_79.tar.gz
• 配置环境变量
vi /etc/profile

export JAVA_HOME=/usr/java/jdk1.7.0_79
export PATH=$JAVA_HOME/bin:$PATH
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar

source /etc/profile

2. 安装tomcat
• 解压tomcat到对应的规范化目录下
• war包拷贝解压或不解压到webapps
• 配置端口
vi /etc/sysconfig/profile

-A INPUT -m state --state NEW -m tcp -p tcp --dport 8180 -j ACCEPT

source iptables restart

3. tomcat内存设置
vi /home/lj/edu/web/boss-tomcat/bin/catalina.sh
在第二行添加
JAVA_OPTS='-Xms128m -Xmx512m -XX:PermSize=128m'

tomcat修改端口
vi /home/lj/edu/web/boss-tomcat/conf/server.xml
